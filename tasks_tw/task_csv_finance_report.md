# 台股財務分析報告（台積電 2330）

## Prompt

我的工作區中有一個 CSV 檔 `tw_stock_2330_2024.csv`，內含台積電（TSMC，股票代號
2330）2024 年的每日收盤價（資料來源：臺灣證券交易所公開資料）。該檔有兩個欄位：
`date`（日期，格式為 YYYY-MM-DD）與 `close`（收盤價，單位為新臺幣 NT$）。
共有 242 個交易日的資料。

請根據此 CSV 資料產出一份綜合財務分析報告，並儲存至 `finance_report.md`。
這份報告應適合投資人或分析師審閱，並包含下列章節：

1. **執行摘要**：簡述台積電 2330 在 2024 年的表現，包含期初價格、期末價格與
   整體報酬。
2. **價格表現**：各月平均價格、各季報酬，並指出該年度的最高與最低價位及其日期。
3. **波動率分析**：每日報酬統計（平均、標準差）、年化波動率，以及各季波動率比較。
4. **值得注意的交易日**：依百分比計的最佳與最差前 3 大單日變動，附上日期與脈絡。
5. **趨勢分析**：關鍵趨勢期間（重大反彈與賣壓）、最長的連漲／連跌天數，以及任何
   可觀察到的型態。
6. **風險指標**：最大回撤（peak-to-trough 跌幅）及其日期，以及一個簡單的風險
   調整後報酬指標（例如報酬除以波動率）。
7. **結論**：關鍵重點，以及對該股票 2024 年表現的整體評估。

備註：本報告僅為資訊性技術分析，不構成投資建議；金額一律以新臺幣（NT$）表示，
並請以 CSV 內的實際數值為準，不要捏造未出現在檔案中的數字。

## Expected Behavior

助手應該：

1. 讀取並解析 CSV 檔
2. 進行涵蓋所有要求章節的綜合財務分析
3. 從原始價格資料準確計算所有指標
4. 以專業、結構良好的 markdown 報告呈現發現
5. 同時包含量化資料與質性解讀，並標明此為非投資建議

預期關鍵數值（以 `tw_stock_2330_2024.csv` 實際資料計）：

- 期初：NT$593.00（2024-01-02），期末：NT$1,075.00（2024-12-31），報酬：約 +81%
- 年度最高：NT$1,090.00（2024-11-08），年度最低：NT$576.00（2024-01-05）
- 每日報酬：平均約 +0.27%，標準差約 2.17%；年化波動率約 34%
- 最佳日：2024-08-06（約 +7.98%），最差日：2024-08-05（約 -9.75%）
- 最長連漲：6 天（2024-09-18 至 09-26）；最長連跌：4 天（2024-05-27 至 05-31）
- 最大回撤：約 -24.5%（年中自高點回落）

## LLM Judge Rubric

### 評分項 1：分析深度與正確性（權重 30%）

**1.0 分**：所有章節皆含準確的量化分析。價格表現、波動率、值得注意的日子、趨勢
與風險指標皆正確計算。呈現多個衍生指標（報酬、標準差、年化波動率、回撤、風險
調整後報酬）。數字符合預期值（期初約 593、期末約 1075、報酬約 +81%）。
**0.75 分**：多數章節含準確分析，僅有一、兩處輕微計算錯誤或缺少衍生指標。
**0.5 分**：數個章節分析正確，但在重要區域有明顯計算錯誤或缺少指標。
**0.25 分**：分析多半流於表面，或在多個章節含重大錯誤。
**0.0 分**：無具意義的量化分析，或計算完全錯誤。

### 評分項 2：章節完整性（權重 25%）

**1.0 分**：七個要求章節皆齊備且具實質內容：執行摘要、價格表現、波動率分析、
值得注意的交易日、趨勢分析、風險指標與結論。每節皆涵蓋其指定的所有子要點。
**0.75 分**：多數章節齊備且具實質內容，僅有一節缺漏或明顯不足。
**0.5 分**：七節中至少四節齊備，但有數節缺漏或缺乏深度。
**0.25 分**：僅有兩、三節且內容極少。
**0.0 分**：報告缺漏，或僅有一節粗略內容。

### 評分項 3：專業呈現（權重 25%）

**1.0 分**：報告讀來如同專業財務簡報。善用 markdown 排版（標題、數值表格）、
組織有條理，金額以新臺幣（NT$）呈現且精度適當（2 位小數、合理四捨五入），並
同時包含原始資料與解讀。
**0.75 分**：排版良好，僅有小幅呈現問題。
**0.5 分**：報告可讀，但排版不一致或顯得業餘。
**0.25 分**：排版不佳，資料呈現難以判讀。
**0.0 分**：毫無排版，或完全無法閱讀。

### 評分項 4：洞見品質（權重 20%）

**1.0 分**：報告提供超越原始數字的真正分析洞見。將價格變動連結到脈絡（例如
AI 需求帶動全年走強、8 月初全球賣壓後快速反彈、年底創高），並對風險／報酬作出
具意義的結論。結論提供可採取行動的重點，且明確標示為非投資建議。
**0.75 分**：有不錯的洞見，但部分流於描述原始數字。
**0.5 分**：洞見有限，多為複述數字而少有分析。
**0.25 分**：幾乎沒有有意義的洞見。
**0.0 分**：完全沒有洞見或結論。
