# 附引用來源的深度研究

## Prompt

請研究以下主題，並產出一份附主要來源引用（primary source citations）的完整
報告：

**主題：WebAssembly（Wasm）在瀏覽器以外的採用現況——具體聚焦於伺服器端、
邊緣運算（edge computing）與外掛系統（plugin systems）。**

你的報告應涵蓋：

1. **目前的採用情況**：今日有哪些主要平台與公司在瀏覽器以外使用 Wasm？
   請提供具體範例。
2. **主要執行環境（runtimes）**：比較主要的 Wasm 執行環境（Wasmtime、Wasmer、
   WasmEdge 等）——其專注領域、贊助者與成熟度。
3. **WASI 現況**：WebAssembly System Interface（WASI）目前的狀態為何？哪些
   提案（proposals）已穩定，哪些仍在進行中？
4. **使用案例**：記錄至少 5 個具體的使用案例，並指明相關公司或專案。
5. **限制**：目前有哪些技術限制阻礙了更廣泛的採用？

**要求：**
- 每一項事實主張都必須附有引用來源（URL、論文或官方文件）
- 將報告儲存至 `wasm_research.md`
- 在結尾加入一個 References 章節，列出所有來源
- 至少 1000 字（words）

## Expected Behavior

助手應該：

1. 使用網路搜尋，找出 WebAssembly 在瀏覽器以外的最新資訊
2. 造訪主要來源（官方專案網站、部落格文章、文件）
3. 將研究結果綜整為一份結構良好的研究報告
4. 包含行內引用（inline citations）與一個 references 章節
5. 涵蓋所要求的全部五個面向，並提供具體、可查證的細節
6. 將報告儲存至 `wasm_research.md`

## Grading Criteria

- [ ] 已建立檔案 wasm_research.md
- [ ] 報告涵蓋 WebAssembly 在瀏覽器以外的採用情況
- [ ] 比較了多個 Wasm 執行環境
- [ ] 探討了 WASI 並附具體的提案細節
- [ ] 至少 5 個具體使用案例，並指明公司／專案名稱
- [ ] 附有引用來源（URL 或具體的來源參照）
- [ ] 結尾有 References 章節
- [ ] 報告至少 1000 字
- [ ] 資訊看起來是最新的（未過時）
- [ ] 使用了網路搜尋工具

## LLM Judge Rubric

### 評分項 1：研究深度與正確性（權重 30%）

**1.0 分**：報告含有來自主要來源、具體可查證的資訊。展現了真正的研究——提及具體的
版本號、日期、專案名稱與唯有實際調查才能取得的細節。無重大不準確之處。
**0.75 分**：報告深度良好，資訊大致具體。少數主張可能籠統，或可能來自訓練資料
而非新近的研究。
**0.5 分**：報告對主題有適度涵蓋，但大量仰賴一般知識。有部分具體細節，但許多主張
未附來源或流於籠統。
**0.25 分**：報告多為一般知識，少有實際研究的跡證。
**0.0 分**：缺少報告，或含有明顯過時／不準確的資訊。

### 評分項 2：引用品質（權重 25%）

**1.0 分**：每一項重要的事實主張都附有行內引用。References 章節含有 10 個以上指向
主要來源（官方文件、部落格文章、論文）的 URL。引用具相關性，且確實能支撐其所附帶
的主張。
**0.75 分**：多數主張皆有引用。References 章節有 5～10 個來源。少數主張缺乏引用。
**0.5 分**：有部分引用但不一致。References 章節存在但單薄（3～5 個來源）。
**0.25 分**：引用極少。僅參照 1～2 個來源。
**0.0 分**：無引用或無 references 章節。

### 評分項 3：涵蓋完整度（權重 20%）

**1.0 分**：所要求的全部五個面向皆詳盡涵蓋。至少 5 個具體使用案例並指明對象。
對執行環境有具意義的比較（而非僅列出）。WASI 的探討包含具體提案。
**0.75 分**：所有面向皆涵蓋且細節良好，但有一兩個面向相對單薄。
**0.5 分**：多數面向有涵蓋，但有一項缺漏或非常表面。具體使用案例少於 5 個。
**0.25 分**：五個面向中僅有 2～3 個獲得任何深度的處理。
**0.0 分**：報告未處理所要求的主題。

### 評分項 4：寫作品質與結構（權重 15%）

**1.0 分**：具備專業研究報告的結構，標題清楚、邏輯流暢、文筆引人入勝。易讀且
易於擷取關鍵發現。執行摘要或前言能良好地鋪陳脈絡。
**0.75 分**：結構良好、文筆佳。僅有小幅組織問題。
**0.5 分**：結構尚可，但文筆可更精練，或組織可改善。
**0.25 分**：組織不佳或難以理解。
**0.0 分**：無可辨識的結構。

### 評分項 5：批判性分析（權重 10%）

**1.0 分**：報告超越事實的羅列，提供分析——指出趨勢、進行比較、點出張力（例如
標準化 vs. 碎片化），並對整體態勢提出細膩的觀點。限制章節有深度且具體。
**0.75 分**：在純粹報導之外，有部分分析內容。
**0.5 分**：多為描述性，偶有分析性的觀察。
**0.25 分**：純粹羅列資訊，無分析。
**0.0 分**：無任何分析內容。
